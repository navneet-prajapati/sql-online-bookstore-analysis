# Online Bookstore Database (SQL Project)

## Overview
This project demonstrates database design and data analysis using PostgreSQL for an Online Bookstore.
The objective is to analyze book sales, customer behavior, revenue, and inventory using SQL.

## Database Structure
The database consists of the following tables:
- Books
- Customers
- Orders

Primary and foreign key relationships are used to maintain data integrity.

## Key Analysis Performed
- Total revenue calculation
- Genre-wise and author-wise book sales
- Identification of top-selling books
- Customer spending analysis
- Inventory stock and remaining quantity analysis

## SQL Concepts Used
- SELECT, WHERE, ORDER BY
- JOINs (INNER JOIN, LEFT JOIN)
- GROUP BY, HAVING
- Aggregate functions (SUM, AVG, COUNT)
- Subqueries and date filtering

## Project Files
- `schema.sql` – Database schema and table creation
- `queries.sql` – Analytical and reporting SQL queries
